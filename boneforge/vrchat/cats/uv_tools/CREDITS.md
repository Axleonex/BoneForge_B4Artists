# BoneForge CATS UV Tools Credits

BoneForge UV methods are integrated into the CATS Material Combiner workflow.
This integration reimplements atlas-focused UV method behavior for BoneForge and
does not vendor the full UVToolkit add-on.

## Modification Notice

These BoneForge UV tools are adapted for the BoneForge CATS Material Combiner
workflow. They use BoneForge operator names and are not a drop-in copy of
UVToolkit.
